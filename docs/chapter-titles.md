# HS / PCT Chapter Titles

Canonical WCO Harmonized System chapter titles (01–97). Pakistan Customs Tariff uses the
international HS chapter set; the first 6 digits of every PCT code are WCO-standard.

Chapter titles are **not printed** in the FBR tariff PDF body, so this reference is the
source of truth for `codes` rows at the `chapter` level (`is_synthetic = TRUE`).

Section groupings per https://conseric.pk/hs-codes/. Chapter 77 is reserved/unused in HS
and absent from the tariff. The tariff covers chapters 01–97.
